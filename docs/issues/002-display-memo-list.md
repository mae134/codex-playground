# Issue002

## Title

Display Memo List

## Goal

固定のメモ一覧を画面に表示できるようにする。

## Scope

以下を実施する。

* 固定のメモデータを用意する
* メモ一覧を画面に表示する
* `map()` を使用して一覧表示する
* 各メモを見やすいカード形式で表示する

## Out of Scope

以下は今回実装しない。

* メモの追加
* メモの削除
* メモの編集
* localStorage
* データベース
* 検索機能
* 並び替え

## Constraints

* AGENTS.mdのルールに従うこと
* Scope外の変更は行わないこと
* 固定データを使用すること
* `map()` を利用して一覧表示すること
* 不明点がある場合は、実装前に前提を説明すること

## Acceptance Criteria

以下を満たした場合、本Issueは完了とする。

* 固定のメモデータが表示される
* `map()` を使用して一覧表示している
* 開発サーバーで正常に表示される
* コードにエラーがない

## Deliverables

以下を成果物として提出する。

* 変更したファイル一覧
* 実行したコマンド
* 動作確認結果
* 必要に応じた改善提案（最大3件）
